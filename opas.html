Määrittely

Animaatio määritellään ensin elementin säännössä. Tämän jälkeen animaation pisteet määritellään @keyframes-säännön avulla.

Nimi-elementin lisäksi määritetään nyt toistojen määrä (infinite) sekä asetetaan suunta vaihtumaan kierrosten välillä.


.animoitu {
      /* Animaation nimi */
      animation-name: testi; 

      animation-duration: 2s;
      animation-iteration-count: infinite; 
      animation-direction: alternate;
}

/* Samalla nimellä keyframes-sääntö: */
@keyframes testi {
      from {
            color:red;
      }

      to {
            color:blue;
      }
}
Hiipsis (class="animoitu")

Useampi avainpiste

Monimutkaisempia animaatioita voidaan määritellä useammilla avainpisteillä:


.animoitu2 {
      /* Animaation nimi */
      animation-name: testi2; 

      animation-duration: 4s;
      animation-iteration-count: infinite; 
      animation-direction: alternate;
}

/* Samalla nimellä keyframes-sääntö: */
@keyframes testi2 {
      from {
            color:red;
      }

      25% {
            color:white;
      }

      50% {
            color:black;
      }

      75% {
            color:white;
      }

      to {
            color:blue;
      }
}
Toinen piip

Animoitavat elementit

Lista kaikista animoitavista elementeistä löytyy esimerkiksi MDN:stä.

Esimerkiksi transform-tyylit sopivat erityisen hyvin käytettäväksi animaatioiden kanssa.


.animoitu3 {
      /* Animaation nimi */
      animation-name: testi3; 

      animation-duration: 2s;
      animation-iteration-count: infinite; 
      animation-direction: alternate;
}

/* Samalla nimellä keyframes-sääntö: */
@keyframes testi3 {
      from {
            transform: rotate(0deg);
      }

      to {
            transform: rotate(360deg);
      }
}
Kolmas piip

Sananen selaintuesta

Käytännösssä edellä mainitut säännöt eivät riitä, sillä osa selaimista ei vielä tue animaatioita ilman selainkohtaisia etuliitteitä.

CanIUse-sivuston perusteella näyttäisi riittävän että normaalien määritysten lisäksi ne kannattaa toistaa -webkit-etuliitteillä, kuten olen tällä sivulla tehnyt. 


.animoitu {
      animation-name: testi; 
      animation-duration: 2s;
      animation-iteration-count: infinite; 
      animation-direction: alternate;
      -webkit-animation-name: testi; 
      -webkit-animation-duration: 2s;
      -webkit-animation-iteration-count: infinite; 
      -webkit-animation-direction: alternate;
}

@keyframes testi {
      from {
            color:red;
      }

      to {
            color:blue;
      }
}

@-webkit-keyframes testi {
      from {
            color:red;
      }

      to {
            color:blue;
      }
}
